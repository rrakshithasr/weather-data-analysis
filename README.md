# 🌦️ Weather Data Analysis Dashboard

## 📌 Project Overview

The Weather Data Analysis Dashboard is a beginner-friendly data visualization project built using **Python**, **Pandas**, and **Matplotlib**. It analyzes weekly weather data and presents key insights through multiple charts in a single dashboard.

This project demonstrates how to load, analyze, and visualize data using different types of plots.

---

## 🚀 Features

- Create and read a CSV dataset
- Analyze weekly weather data
- Calculate weather statistics
- Generate a dashboard with six different charts
- Save the dashboard as an image

---

## 📊 Visualizations

- 🌡️ Temperature Line Chart
- 💧 Humidity Bar Chart
- 🌧️ Temperature vs Rainfall Scatter Plot
- 🥧 Rainfall Distribution Pie Chart
- 🌬️ Wind Speed Histogram
- 📈 Temperature & Humidity Comparison Chart

---

## 📈 Analysis Performed

- Average Temperature
- Highest Temperature
- Lowest Temperature
- Average Humidity
- Total Rainfall
- Maximum Wind Speed
- Hottest Day
- Rainiest Day


   - Analyze the data
   - Display the dashboard
